## About This Book

This book teaches you how compilers work — from scanning source code, parsing grammar, performing semantic analysis, generating intermediate representations, and finally producing machine code — all using the LLVM compiler infrastructure. Each chapter includes both theoretical foundations and hands-on code examples.

## Prerequisites

- Proficiency in C++ (the primary language for LLVM)
- Basic understanding of data structures (trees, graphs, hash tables)
- Familiarity with command-line tools and build systems (CMake)

## How to Use This Book

Read the chapters in order. Each chapter builds on the previous one. Code examples are self-contained and can be compiled and run with LLVM installed on your system.

## LLVM Version

The code examples target **LLVM 17+** and use the new pass manager and opaque pointer types.
